# YOLO + MongoDB + Flask Integration

Lightweight Flask app for YOLO detection with MongoDB backend. Designed for quick deployment and testing.

## Features
- YOLO detection
- Flask REST API
- MongoDB storage
- Easy to run and extend

## Tech Stack
- YOLO (Ultralytics)
- Flask
- MongoDB
- Python

## Use Case
Perfect for prototyping and edge-based CV applications.

   cd integeration_YOLO_with_MongoDB_and_flask
